# Proposition
- a declaration that is either true or false
- e.g. "1+1=2"
	- value is true
- "1+1=3"
	- value is false
- NOT propositions
	- "1+1>X"
	- its not a proposition as we dont know if its true or not

# Paradox
- a declaration that cannot be assigned a truth value
- "This statement is false"
